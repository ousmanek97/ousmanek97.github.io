## 這週學了什麼

用 [Astro](https://astro.build/) 架 static site 比我想像中容易很多。整個流程：

1. `npm create astro@latest` 選 blog template
2. 改 `consts.ts` 設定網站標題
3. 用 Markdown 寫文章
4. Push 到 GitHub，GitHub Actions 自動 deploy

整個過程大概花了一個小時。

## 這週的想法

開始寫週記的動機很簡單：想要一個地方記錄自己「有沒有持續在學東西、做有意義的事」。

比起看 streak 或 habit tracker，公開寫出來的壓力感剛剛好。不會太重，但夠讓我認真對待。
